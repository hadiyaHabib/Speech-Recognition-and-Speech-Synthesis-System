Voice Assistant using Python
Project Description
This project is a personalized voice assistant developed using Python. It can understand voice or text input from the user, process predefined queries, and respond through both text and speech. The assistant supports male and female voice options and is mainly designed for educational and personal use.

Features
Voice to Text conversion (Speech Recognition)
Text to Speech response (Male/Female voice)
Predefined knowledge base (50+ common queries)
Interactive Gradio-based user interface
Supports both voice and text input
Offline functionality (no live internet data)

Tools & Technologies Used
Python
SpeechRecognition
gTTS
pydub
Gradio
datetime

How It Works
User provides input via voice or text.
Voice input is converted into text using SpeechRecognition.
The system matches the input with a predefined dataset.
A suitable text response is generated.
The response is converted into speech using gTTS.
Output is displayed and played through the Gradio interface.
How to Run the Project

Clone the repository:
git clone https://github.com/your-username/voice-assistant-python
Install required libraries:
pip install speechrecognition gtts pydub gradio

un the Python file:
python app.py
Open the link generated by Gradio in your browser.
